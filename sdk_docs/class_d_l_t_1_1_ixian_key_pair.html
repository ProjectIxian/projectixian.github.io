Detailed Description

Represents the raw data of an Ixian RSA key value.

The Key Pair includes the RSA public and private keys, serialized from an internal format to make them easier to use in network communications. In addition, the IxianKeyPair structure also includes the 'nonce' value, which is used to easily and quickly generate new Ixian addresses without having to recompute new RSA keys.
